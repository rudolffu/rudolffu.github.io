    Useful tools/links for observations

    1. Object visibility

    • Object Visibility:
      http://catserver.ing.iac.es/staralt/

    2. Standard star catalogs

    • List of standard star catalogs:
      https://www.eso.org/sci/observing/tools/standards.html

    • RA Ordered List of Spectrophotometric Standards:
      https://www.eso.org/sci/observing/tools/standards/spectra/stanlis.html

    3. Finding charts

    • SDSS DR16 Finding Chart Tool:
      http://skyserver.sdss.org/dr16/en/tools/chart/chartinfo.aspx

    • IRSA Finder Chart Service:
      https://irsa.ipac.caltech.edu/applications/finderchart/

    • PanSTARRS-1 Image Access:
      http://ps1images.stsci.edu/cgi-bin/ps1cutouts

    • Python API for getting image from the PS1 image server:
      https://ps1images.stsci.edu/ps1image.html

    • panstamps – A python package and command-line tools to download stacked and/or warp image stamps from the STScI PanSTARRS image server.
      https://panstamps.readthedocs.io/en/latest/index.html
      https://github.com/thespacedoctor/panstamps
